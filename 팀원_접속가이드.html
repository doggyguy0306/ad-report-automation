🍎 macOS

서버_시작.command

더블클릭하면 터미널 창이 열리고
접속 주소가 표시됩니다.

🪟 Windows

서버_시작.bat

더블클릭하면 cmd 창이 열리고
접속 주소가 표시됩니다.

💡 실행 후 터미널 창에 표시된 IP 주소를 팀원들에게 공유하세요.
예: http://192.168.1.5:8501
⚠️ 터미널 창을 닫으면 서버가 꺼집니다. 팀원들이 사용하는 동안은 창을 열어두세요.

문제 해결

증상해결법
팀원이 접속이 안 된다 ① 서버 PC가 켜져 있는지 확인
② 같은 와이파이/사내망인지 확인
③ 방화벽에서 8501 포트 허용
macOS에서 .command 실행 거부 파일 우클릭 → 열기 → 확인 클릭
"streamlit 명령어를 찾을 수 없음" 최초설치 파일을 다시 실행하거나
pip3 install streamlit 실행
보고서 생성 중 오류 파일명에 키워드가 포함됐는지 확인
(주간리포트, 게재된 시점, 전환 등)
